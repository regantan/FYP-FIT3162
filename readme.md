## Setup & Installation

1. Clone repository 
``` git clone https://github.com/regantan/FYP-FIT3162.git```
2. Install requirements
``` 
cd FYP-FIT3162
pip install -r requirements.txt
```
3. Training of model
```
cd pyabsa/absa_model
python model_train.py
```
